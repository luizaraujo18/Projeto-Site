Aprendendo Python

Em outubro de 2021, a tecnologia Python assumiu o posto de linguagem de programação mais usada no mundo. O feito merece respeito: essa é a primeira vez que isso acontece em 20 anos, desbancando Java e C, as linguagens que até então reinavam supremo na lista organizada pelo site Tiobe. Essa conquista apenas reforça o que é Python: um dos tipos de linguagem de programação mais fácil de aprender e que proporciona uma excelente oportunidade de carreira. A tecnologia está intimamente ligada a diversas carreiras, como Data Science, desenvolvimento web e de aplicativos, automação de scripts e até no setor de finanças.
